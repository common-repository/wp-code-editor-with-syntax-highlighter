This wordpress plugin converts the default plain text theme and plugins editor in wordpress to a pretty code editor. 

== Description ==
WP Code Editor with Syntax Highlighter is a must have plugin if you hate the plain text code editor in the wordpress admin panel. It converts it to a beautify code friendly editor and saves you any stress in using an external code editor.

Features:
* Supports Xml, php, css, html, javascript
* Matching tag functions
* Matching bracket functions. 
* Code indentation.
* Line numbering.

== Installation ==
1. Upload WP Code Editor with Syntax Highlighter to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
